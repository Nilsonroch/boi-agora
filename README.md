# Boi Agora

Aplicativo web com atualização em tempo real para pecuária de corte.

## Recursos
- Preço da arroba
- Mercado futuro do boi gordo
- Grãos por praça
- Radar de notícias
- Filtro por estado
- Alertas de alta e baixa com base no histórico local
- Mini gráficos de evolução salvos no navegador
- Atalho para WhatsApp
- Instalação como app no celular via PWA

## Desenvolvimento
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Publicação no Netlify
- Publicar o projeto completo no Netlify
- Pasta de publicação: `dist`
- Funções: `netlify/functions`
